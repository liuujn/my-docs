# 欢迎使用 VitePress

这是你的第一个 Markdown 页面！

## 功能特性

- ✅ 支持代码高亮
- ✅ 支持表格、列表、图片等
- ✅ 响应式设计

## 代码示例

```js
console.log('Hello VitePress!')

const users = [
    { name: "Alice", age: 25 },
    { name: "Bob", age: 30 }
]

const adults = users.filter(user => user.age >= 18)
console.log(adults)
```
